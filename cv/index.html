CodeRunner
远程代码执行平台

基于 FastAPI + React 18 构建的现代化代码执行平台，采用 JWT + Argon2 安全认证和四级用户权限体系。核心技术栈包括 FastAPI 异步框架、Ant Design UI 组件库、Docker 容器化部署和 Conda 环境管理。实现了安全沙箱代码执行、AI 智能代码生成（支持多种 AI 模型）、个人代码库管理和社区交互系统，具备完整的用户管理、日志审计和资源配限功能。项目采用单体架构设计，支持高并发处理和多租户隔离，展现了全栈开发和系统架构设计的综合能力。

WebMonitor
智能网页内容监控与多渠道通知平台

基于 FastAPI + React 18 构建的现代化网页内容监控平台，采用 JWT + bcrypt安全认证和基于角色的用户权限体系（管理员/普通用户）。核心技术栈包括 Selenium WebDriver 动态内容抓取、APScheduler任务调度、Material-UI 组件库和 SQLAlchemy ORM 数据持久化。实现了精准的 XPath选择器内容提取、智能变更检测算法、多邮件配置 SMTP 通知系统和任务订阅分享机制，具备完整的用户管理、监控日志审计、域名黑名单过滤和实时任务监控功能。项目采用前后端分离架构设计，支持 SQLite/PostgreSQL 数据库切换、无头 Chrome浏览器自动化和高并发任务处理，展现了全栈开发、Web自动化爬虫技术和系统架构设计的综合能力。
