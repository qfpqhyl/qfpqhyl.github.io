建站之路：绕了一圈回到原点

这几年折腾过不少建站工具。大一刚开始用的是 Hexo，但总感觉美观度不够、主题选择有限、自定义能力也受限。后来尝试过 Hugo、WordPress 等工具，可 Hugo 还需要配置服务器，WordPress 又老是提示插件不安全。

最后还是回到了 Hexo。至少可以直接部署到 GitHub Pages，不用担心安全问题，再大的流量和攻击都不怕宕机，也不用长时间购买国外 VPS（国内的 VPS 绑定域名还得备案）。重新用 HexO 搭建网站，有种熟悉的感觉。

本周动态

学习充电

最近准备开始 Java 项目开发，把 Java 基础知识重新巩固了一遍。跟着 yupi 的教程 学习，教程内容很扎实，配合 Dia 浏览器的侧边栏 AI 助手，遇到不懂的地方随时提问，学习效率提升了不少。

生活点滴

研究生数学建模比赛出成绩了，拿到了三等奖。说实话，比赛刚开始几天就后悔参加了，但学校规定只有成功参赛才能报销报名费，为了报名费也只能硬着头皮把代码写完，论文部分就交给队友负责了。正如陈粒歌里唱的「无所求便满载而归」，虽然三等奖不算理想的成绩，但考虑到自己投入的时间和精力有限，结果还算满意。

创意清单

  • 学习 Go 语言开发终端工具，比如一个增强版的 SSH 工具，具体功能还在构思中
  • 重构网页变化监控项目，改为前后端分离架构，放在简历上展示「已完成，项目链接」
